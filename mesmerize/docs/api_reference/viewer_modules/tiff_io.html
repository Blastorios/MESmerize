Tiff Module¶

Viewer module used for loading tiff files into the Viewer Work Enviroment. Can also be used with scripts within Mesmerize for loading tiff files without using the more back-end API of mesmerize.viewer.core.ViewerWorkEnv and mesmerize.viewer.core.ViewerUtils

class mesmerize.viewer.modules.tiff_io.ModuleGUI(parent, viewer_reference)[source]¶
load_tiff_file(tiff_path, meta_path, method)[source]¶

Load a tiff file along with associated meta data

Parameters:
  • tiff_path – path to the tiff file
  • meta_path – path to the json meta data file
  • method – one of “asarray”, “asarray-multi”, or “imread”. “asarray” and “asarray-multi” use tifffile.asarray(), “asarray-multi” is for multi-page tiffs, and “imread” uses tifffile.imread()
