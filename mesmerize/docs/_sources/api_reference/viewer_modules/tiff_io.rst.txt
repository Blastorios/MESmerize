Tiff Module
***********

Viewer module used for loading tiff files into the Viewer Work Enviroment. Can also be used with scripts within Mesmerize for loading tiff files without using the more back-end API of :class:`mesmerize.viewer.core.ViewerWorkEnv` and :class:`mesmerize.viewer.core.ViewerUtils`
